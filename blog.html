What is Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents.

How do you selecr an element from DOM?

What is event delegation in the context of DOM and why is it useful?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element, and that listener handles events triggered by its child elements.

How do you manipulate an element's attribute and styles using this DOM?

You can manipulate an element's attributes and styles using the DOM in JavaScript by interacting with specific properties and methods of the element.
